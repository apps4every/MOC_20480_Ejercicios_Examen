Question 113 You develop an HTML5 application that allows users to upload files from their local computers. The user interface must remain responsive during the upload. You need to implement the file upload functionality for the application. Which two actions should you perform? (Each correct answer presents a complete solution. Choose two.) A. Use an HTML form with a file type INPUT element that targets a hidden IFRAME element. B. Use a file type INPUT element, and then use the Web Storage API to upload the file. C. Use a FormData object and upload the file by using XMLHttpRequest. D. Register the file protocol by using protocol handler registration API and then upload the file by using XMLHttpRequest. E. Use the FileSystem API to load the file, and then use the jQuery post method to upload the file to the server.
Correct answer: BD

Comprobación:

Se necesita usar funciones asíncronas, es decir que no dejen al usuario en espera de finalizar la operación de enviar el fichero.

La opciones válidas serían B: "Web Storage API" y D: "XMLHttpRequest"

Referencia:

https://www.w3schools.com/js/js_ajax_http.asp

Sin embargo también podría ser solución la C, según la siguiente referencia: https://developer.mozilla.org/en-US/docs/Web/API/FormData/Using_FormData_Objects

En el siguiente enlace se indica como usar XMLHttpRequest para mostrar el progreso de la petición al servidor:https://developer.mozilla.org/en-US/docs/Web/API/XMLHttpRequest/Using_XMLHttpRequest
