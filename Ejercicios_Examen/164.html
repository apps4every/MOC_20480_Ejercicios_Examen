Question 164 You are validating user input by using built-in JavaScript functions. The application must: · Store the value that is entered in a variable named inputValue · Use the built-in isNaN(tnputValue) function to evaluate the data type You need to validate the return value of the isNaN(inputValue) function. Which values will be returned? isNaN (-13) isNaN (24.3) isNaN (3*8) isNaN ('5')

Respuesta correcta: Todas las funciones isNan devuelve false ya que los valores que se le pasan son correctos.

La función isNaN devuelve true cuando el valor que se la pasa es un número inválido. '5' es igual a 5.
