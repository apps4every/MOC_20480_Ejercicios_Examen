Question 136 You develop an HTML5 webpage that contains the following HTML markup: <label for="loanTypes"Loan Types></label> <select name ="loanTypes" id="loanTypes" multiple="multiple"> <option>Secured</option> <option selected="selected">Unsecured</option> <option>Demand</option> <option selected="selected">Subsidized</option> </select> The page includes the fllowing Javascript code: var selectedValues=''; You have the following requirements: * Retrieve the content for any OPTION elements when the value of the selected attribute equals selected. * Add the content to a comma separated string. You need to retrieve the required data. Which two code segments should you add to the webpage? Each correct answer presents a complete solution. NOTE: Each correct selection is worth one point.
Correct answer: C y D
Opción C: $("select option:selected").each(function() { selectedValues += $(this).text() + ','; }); Opción D: $("#loanTypes option:selected").each(function() { selectedValues += $(this).text() + ','; });
La A no es, usa checked en lugar de select que sería para otro tipo de campos, checkbox.
La B no es, al usar el método each necesita usar this para obtener el valor de la propiedad text, en su lugar usar el propio objeto loanTypes.
