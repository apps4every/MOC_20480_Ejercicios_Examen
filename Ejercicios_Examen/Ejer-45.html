Ejercicio No 28

You are developing an HTML5 page. The page includes the following code.

The inner paragraph must be exactly 15 pixels from the top left corner of the outer paragraph. You set the left style for the inner paragraph to the appropriate value. You need to set the position property of the inner paragraph. Which value should you use?

  • absolute
  • static
  • fixed
  • relative

Correct Answer: A

Correct Answer: A Section: Volume A Explanation Explanation/Reference: Explanation: absolute: The element is positioned relative to its first positioned (not static) ancestor element. Incorrect Answers: D: relative: The element is positioned relative to its normal position, so "left:20" adds 20 pixels to the element's LEFT position.
